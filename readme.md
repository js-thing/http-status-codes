> This library is a complete collection of all Http Status Codes for Hypertext Transfer Protocol (HTTP/1.1) defined in RFC 7231.

## Installation

```sh
# Locally in your project.
npm install abc

# Or globally with TypeScript.
npm install -g abc
```
